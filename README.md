# 简单后门病毒分析

## 样本介绍

除窃取信息类型病毒外,其他类型病毒具有一定破坏性,网上公开源码较少,不可控,分析难度较大,不适宜教学使用。所以本文选择编写一个简单的后门程序,先不看其源码,编译之后,对二进制文件进行行为分析、抓包分析和逆向分析,最后总结结论再与源码进行比较。
